    Overview #

    Paimon not only supports Flink SQL writes and queries natively, but also provides queries from other popular engines, such as Apache Spark and Apache Hive.

    Compatibility Matrix #

    Doris 2.0+ ✅ ❌ ❌ ❌ ❌ ❌ ❌

    Recommended versions are Flink 1.17.2, Spark 3.5.0, Hive 2.3.9
